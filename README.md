# SnippetDeck Pro 🚀

**SnippetDeck Pro** is a high-performance Chrome Extension designed for technical presenters and developers. It allows you to store, tag, and quickly copy multi-line code snippets or talking points during live demos.

## ✨ Key Features

- **Multi-Category Tagging:** Organize snippets by `AI`, `Prompt`, `Sitecore`, `Codelab`, and more.
- **Advanced Filtering:** Multi-select categories to narrow down exactly what you need.
- **Quick Copy:** One-click copy to clipboard with visual feedback.
- **Favorites:** Pin your most important "Plan B" snippets to the top.
- **Search:** Real-time search across titles and content.
- **CRUD Operations:** Full administrative interface to add, edit, or delete snippets.

## 🛠️ Tech Stack

- **Framework:** React 19 (TypeScript)
- **Bundler:** Vite
- **Icons:** Lucide-React
- **Styling:** Vanilla CSS (Modern CSS Variables)

## 📥 Installation

1. Clone the repository.
2. Run `npm install`.
3. Run `npm run build`.
4. Open Chrome and navigate to `chrome://extensions/`.
5. Enable **Developer mode** (top right).
6. Click **Load unpacked** and select the `SnippetDeck/dist` folder.
